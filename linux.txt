Linux Commands

Commands                                           Syntax                                Usage 

1) pwd                                             pwd                                   it will print the working directory

2)whoami                                           whoami                                it will print the username of the logged in user

3) mkdir                                           mkdir brachname                       it will create the directory with the name of entered name

4) cd                                              cd path_to_ directory                 it will change the current working directory

5) cd ..                                           cd ..                                 it will change to the parent directory of the working directory

6) cd .                                            cd .                                  it will represent to the current directory

7) ls                                              ls                                    it will give the list of directories  and files inside the current 
                                                                                         working directory

8)rmdir                                            rmdir directory_ name                 it will remove empty directory only

9) touch                                           touch filename                        it will create the file with specified name

10) rm                                             rm filename                           it will remove the specified file'

11) rm                                             rm -r directory name                  it will remove the specified directory

12) cp                                             cp path1/file1 path2                  it will copy the file1  from path1 to path2 

13) mv                                             mv path1/file1 path2                  it will move the file1  from path1 to path2 

14) cat                                            cat filename                          it will show the content of the specified file on the console file editors

15) nano                                           nano filename                         it will opens the file name for the writing  
                                                                                         control + o = it will save the file and prompt for the filename press enter

16) vi                                             vi filename                           this opens file for writing 
                                                                                         press i for insert mode
                                                                                         write the contents 
                                                                                         press esc to come out of insert mode 
                                                                                         press :wq  to save and come out the quit

17) cat                                            cat>filename                          this will open the file in writting mode and it will overwrite the existing content

18) cat                                            cat>>filename                         this will append the new content with existing old content
